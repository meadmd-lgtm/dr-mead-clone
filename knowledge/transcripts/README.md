# Transcripts Folder

Place video and audio transcript files here.

## Naming Convention

```
YYYY-MM-DD_topic-slug.md
```

Examples:
- `2024-03-15_thyroid-and-weight-loss-webinar.md`
- `2024-06-01_patient-qa-gut-health.md`
- `2025-01-10_instagram-live-hormones.md`

## Format for Each Transcript File

```markdown
# [Title of Video/Podcast/Webinar]

**Date:** 
**Platform:** [YouTube / Podcast / Instagram / Webinar]
**Link (if public):**
**Duration:**
**Topics covered:**
```
